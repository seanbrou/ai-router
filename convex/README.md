# Convex Scaffold

This repo uses Convex for add-on profiles, provider settings, install tokens, and ranking logs.

## Important

Run `npx convex dev` before adding real Convex functions. That command generates `convex/_generated`, including the AI guidelines file referenced by the project instructions.
